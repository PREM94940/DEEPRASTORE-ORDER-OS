# Deeprastore Post-Pilot Backlog

The following items have been identified during the pre-pilot Reality Audit and are slated for implementation *after* the 7-day internal pilot.

## High Priority (V2.1 Features)
- **Raise Exception UI**: Add a dedicated button and modal (accessible from Customer 360 and Order details) for staff to raise Exceptions directly, triggering `createExceptionAction`.
- **Add Lead UI**: Add an entry point in the Customer 360 Leads tab to manually input and track new CRM leads.
- **Workflow Discoverability**: Introduce a global "Quick Actions" dashboard or floating action button (FAB) for standard workflows (New Order, Record Payment) so they are not exclusively hidden behind the Global Search bar.
